# Mezan

تطبيق تخطيط مالي محلي يعمل مباشرة في المتصفح، باللغتين العربية والإنجليزية.

## الملفات

- `index.html`: الصفحة الأساسية.
- `css/variables.css`: ألوان ومتغيرات التصميم.
- `css/layout.css`: التصميم العام والمكوّنات الأساسية.
- `css/responsive.css`: قواعد التوافق مع الموبايل والكمبيوتر.
- `js/app.js`: تدفق الشاشات وربط واجهة المستخدم.
- `js/storage.js`: التحقق من البيانات، الترحيل، والاستيراد والتصدير المحلي.
- `js/calculations.js`: تنظيف الأرقام والحسابات المالية.
- `js/translations.js`: أدوات اختيار اللغة واتجاه الصفحة.
- `js/currency.js`: العملات وتنسيق المبالغ.
- `js/ui.js`: تعقيم النصوص والتاريخ المحلي.
- `tests.html`: اختبارات وحدات خفيفة تعمل في المتصفح.

## التشغيل

افتح `index.html` مباشرة بالمتصفح، أو ارفعه على GitHub Pages.

## الاختبارات

افتح `tests.html` في المتصفح. تظهر النتيجة باللون الأخضر عند نجاح جميع الاختبارات.
