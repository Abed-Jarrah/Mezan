# Mezan

نسخة مقسمة من تطبيق ميزان.

## الملفات

- `index.html`: الصفحة الأساسية.
- `css/variables.css`: ألوان ومتغيرات التصميم.
- `css/layout.css`: التصميم العام والمكوّنات الأساسية.
- `css/responsive.css`: قواعد التوافق مع الموبايل والكمبيوتر.
- `js/app.js`: منطق التطبيق الحالي بالكامل.
- ملفات `js/storage.js`, `calculations.js`, `translations.js`, `currency.js`, `ui.js`: جاهزة للتقسيم الأعمق لاحقاً بدون كسر النسخة الحالية.

## التشغيل

افتح `index.html` مباشرة بالمتصفح، أو ارفعه على GitHub Pages.
